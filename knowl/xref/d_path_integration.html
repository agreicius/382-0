Definition 1.12.8. Integration of complex-valued functions.

Let \(f\colon [a,b]\rightarrow \C\) be a complex-valued function, and write \(f=u+iv\text{.}\) We say \(f\) is Riemann integrable on \([a,b]\) if the Riemann integrals
\begin{align*} \int_a^b u(t)\, dt \amp \amp \int_a^b v(t)\, dt \end{align*}
exist, in which case we write
\begin{equation*} \int_a^b f(t)\, dt=\int_a^bu(t)\, dt+i\int_a^b v(t)\, dt\text{.} \end{equation*}
