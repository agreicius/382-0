Lemma 1.25.4.

Let \(f\) be a complex function with open domain \(D\text{.}\) The following statements are equivalent.
  1. \(f\) is open.
  2. For all open subsets \(U\subseteq D\) and all \(z_0\in U\) there is an open ball \(B_r(z_0)\subseteq U\) such that \(f(B_r(z_0))\) is open.

Proof.

We prove both implications separately.
Assume \(f\) is open, let \(U\) be an open subset of \(D\text{,}\) and let \(z_0\in U\text{.}\) Since \(U\) is open, we have \(B_r(z_0)\subseteq U\) for some \(r> 0\text{.}\) Since \(f\) is open, \(f(B_r(z_0))\) is open.
Now assume (2) holds, and let \(U\) be an open subset of \(D\text{.}\) Given any \(w_0\in f(U)\text{,}\) we have \(w_0=f(z_0)\) for some \(z_0\in V\text{.}\) By condition (2), there is an open ball \(B_r(z_0)\subseteq U\) such that \(V=f(B_r(z_0))\) is open. Since \(w_0\in f(B_r(z_0))\subseteq f(U)\text{,}\) we have shown that for all \(w_0\in f(U)\) there is an open set \(V\) such that \(w_0\in V\subseteq f(U)\text{.}\) It follows that \(f(U)\) is open.
