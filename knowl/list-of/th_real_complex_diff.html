Theorem 1.12.8. Complex and path differentiation.

Let \(f\colon U\rightarrow \C\) be holomorphic, and suppose \(\phi\colon [a,b]\rightarrow U \) is a smooth path.
  1. The path \(\psi=f\circ \phi\) is smooth on \([a,b]\) and satisfies \(\psi'(t)=f'(\phi(t))\phi'(t)\) for all \(t\in [a,b]\text{.}\)
  2. In particular, if \(U\) contains the real interval \([a,b]\text{,}\) then the path \(\psi\colon [a,b]\rightarrow \C\) defined as
    \begin{equation*} \psi(t)=f(t) \end{equation*}
    is smooth and satisfies \(\psi'(t)=f'(t)\text{.}\)

Proof.
