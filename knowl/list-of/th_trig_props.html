Theorem 1.7.9. Cosine and sine properties.

Let \(z,w\in \C\text{.}\)
  1. \(\cos z=\cos w\) if and only if \(z=w+2\pi k\) for some \(k\in \Z\text{,}\) or \(z=-w+2\pi k\) for some \(k\in \Z\text{.}\)
  2. \(\sin z=\sin w\) if and only if \(z=w+2\pi k\) for some \(k\in \Z\) or \(z=\pi-w+2\pi k\) for some \(k\in \Z\text{.}\)

Proof.
