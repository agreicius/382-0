Theorem 1.10.10. Functions of derivative zero.

Let \(f\) be defined on the open set \(U\text{.}\) The following statements are equivalent.
  1. \(f\) is locally constant on \(U\text{.}\)
  2. \(f'(z)=0\) for all \(z\in U\text{.}\)
As a consequence, if we assume further that \(U\) is connected, then \(f'\vert_U=0\) if and only if \(f\) is a constant function.
