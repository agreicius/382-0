Lemma 1.25.4.

Let \(f\) be a complex function with open domain \(D\text{.}\) The following statements are equivalent.
  1. \(f\) is open.
  2. For all open balls \(B_r(z_\) sets \(U\subseteq D\) and element \(z_0\in U\) there is an \(r> 0\) such that \(B_r(f(z_0))\subseteq f(U)\text{.}\)

Proof.

The equivalence of the two statements follows immediately from the definition of open: namely, that \(f(U)\) is open if and only if for all \(w_0=f(z_0)\in f(U)\) we there is an \(r> 0\) such that \(B_r(w_0)\subseteq f(U)\text{.}\)
