Theorem 1.25.8. Inverse function theorem.

Assume \(f\) is analytic at \(z_0\) and that \(f'(z_0)\ne 0\text{.}\) There is an open ball \(B=B_r(z_0)\) satisfying the following properties.
  1. Let \(V=f(B_r(z_0))\text{.}\) The function \(f\colon B_r(z_0)\rightarrow V\) is bijective.
  2. The inverse map \(f^{-1}\colon V\rightarrow B_r(z_0)\) is analytic and satisfies
    \begin{align*} (f^{-1})'(w) \amp = \frac{1}{f'(f^{-1}(w))} \end{align*}
    for all \(w\in U\text{.}\)
  3. For all \(w\in V\) we have
    \begin{equation} f^{-1}(w) =\frac{1}{2\pi i}\int_{\gamma}\frac{zf'(z)}{f(z)-w}\, dz, \ \gamma(t)=z_0+re^{it}, t\in [0,2\pi]\text{.}\tag{1.97} \end{equation}

Proof.
