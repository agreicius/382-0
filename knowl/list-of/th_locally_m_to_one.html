Theorem 1.25.5. Locally \(m\)-to-one.

Let \(f\) be a nonconstant analytic function with open domain \(D\text{.}\) Fix any open subset \(U\subseteq D\) and \(z_0\in U\text{.}\) Let \(w_0=f(z_0)\text{,}\) let \(h(z)=f(z)-w_0\text{,}\) and let \(m\) be the order of \(z_0\) as a zero of the function \(f(z)-w_0\text{.}\) There is an open ball \(B_r(z_0)\subseteq U\) satisfying the following properties.
  1. For all \(z\in B_r(z_0)-\{z_0\}\text{,}\) \(f(z)\ne w_0\text{.}\)
  2. For all \(w\ne w_0\in f(B_r(z_0))\text{,}\) the function \(h(z)=f(z)-w\) has \(m\) distinct zeros in \(B_r(z_0)\text{,}\) each of order one. In particular, \(f\) is an \(m\)-to-\(1\) function when restricted to punctured open ball \(B_r(z_0)-\{z_0\}\text{.}\)
  3. \(f(B_r(z_0))\) is open.

Proof.
