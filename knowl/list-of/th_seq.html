Theorem 1.5.2. Limit properties.

  1. \(z_{n}\rightarrow z\) iff \(\val{z_n-z}\rightarrow 0\text{.}\) In particular, \(z_{n}\rightarrow 0\) iff \(\val{z_n}\rightarrow 0\text{.}\)
  2. A sequence \(\{z_{n}\}\) is convergent if and only if \(\{\Re z_{n}\}\) and \(\{\Im z_{n}\}\) are both convergent, in which case \(\displaystyle\lim_{n\rightarrow\infty}z_{n}=\lim_{n\rightarrow\infty}\Re z_{n}+i\limasto[n]{\infty}\Im z_{n}\text{.}\)
  3. If \(z_{n}\rightarrow z\text{,}\) then \(\val{z_n}\rightarrow \val{z}\text{.}\)
  4. If \(z_{n}\rightarrow z\) and \(w_{n}\rightarrow w\text{,}\) then \((cz_{n}+dw_{n})\rightarrow (cz+dw), z_{n}w_{n}\rightarrow zw, z_{n}/w_{n}\rightarrow z/w\) (provided \(w\ne 0\)).
  5. \(z_{n}\rightarrow \infty\) if and only if \(\frac{1}{z_{n}}\rightarrow 0\text{.}\)

Proof.
