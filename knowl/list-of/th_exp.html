Theorem 1.6.8. Exponential function properties.

  1. We have \(e^{z+w}=e^ze^w\) for all \(z,w\in \C\text{.}\)
  2. We have \(e^z=e^w\) if and only if \(\abs{z}=\abs{w}\) and \(\Im w=\Im z+2\pi k\) for some \(k\in \Z\text{.}\) In particular, \(\exp\) is not injective.
  3. \(\im \exp=\C-\{0\}\text{.}\)

Proof.
