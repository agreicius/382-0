Theorem 1.19.6. Limit point.

Let \(U\) be open and connected, and assume \(f\) is holomorphic on \(U\text{.}\) Let \(Z=\{z\in U\colon f(z)=0\}\) be the set of zeros of \(f\) on \(U\text{.}\) If \(Z\) has an limit point, then \(f(z)=0\) for all \(z\in U\text{.}\)

Proof.

We first show that if \(z_0\) is a limit point of \(Z\text{,}\) then \(\ord_f(z_0)=\infty\text{:}\) i.e., we have \(f^{(n)}(z_0)=0\) for all \(n\geq 0\text{.}\) Since \(z_0\) is a limit point of \(Z\text{,}\) we can find a sequence \((z_n)_{n=1}^\infty\) of elements of \(Z\) such that \(\abs{z-z_n}< 1/n\text{.}\) It follows that \(z_n\to z_0\text{.}\) Since \(f\) is continuous on \(U\text{,}\) it follows that \(f(z_0)=f(\lim\limits_{n\to \infty}f(z_n)=0\text{.}\) Thus \(z_0\) is itself a zero of \(f\text{.}\) Now assume by contradiction that \(\ord_f(z_0)=m\) for some positive integer \(m\text{.}\) By CorollaryÂ 1.19.3, we would then have \(f(z)=(z-z_0)^mg(z)\) where \(g(z_0)\ne 0\text{.}\)
