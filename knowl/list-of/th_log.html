Theorem 1.7.4. Logarithm properties.

  1. \(e^w=z\) if and only if \(w\in \log z\) if and only if \(w=\Log z+2\pi k i\) for some \(k\in \Z\text{.}\)
  2. The following identities are valid when interpreted appropriately as equalities of sets.
    1. \(e^{\log z}=z\) for all \(z\ne 0\text{.}\)
    2. \(\log(z_1z_2)=\log(z_1)+\log(z_2)\) for all \(z_1,z_2\ne 0\text{.}\)
    3. \(\log(z_1/z_2)=\log(z_1)-\log(z_2)\) for all \(z_1,z_2\ne 0\text{.}\)
    4. \(\log(z^{1/n})=\frac{1}{n}\log z\) for all \(z\ne 0\) and \(n\in \Z-\{0\}\text{.}\)
